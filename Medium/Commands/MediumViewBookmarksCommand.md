# medium.bookmarks

## Syntax

```G1ANT
medium.bookmarks
```

## Description


This command allows the User to view his bookmarks on Medium and keep a track on the same.

# Sequence of Operation
1. The Robot opens up Medium in the Web browser and waits for the user to enter the credentials.
2. After Successful authentication, the user is granted permission to login and will be taken to the place for viewing his profile, notifications and finally the bookmarks he has kept on his articles.
## Example

This script opens chrome browser and views bookmarks for the users on Medium

```G1ANT
dialog message ‴Testing Medium for Web broswer‴
medium.open
delay 5
medium.login email ‴hi@gmail.com‴ 
delay 4
medium.notifications
delay 5
medium.viewprofile
delay 5
medium.bookmarks 

```
