# lineandroid.collections

## Syntax

```G1ANT
lineandroid.collections text ⟦text⟧
```

## Description

This command allows the user to create his own collections on LINE Application.

# Sequence of Operation
1. The Robot Opens LINE Application on Android and logs in with the phone number.
2. After successful authentication, the Robot guides the user to the Collections section where he/she can create their own set of collection. 
## Example

This simple script opens LINE Application on Android Phones and ensures that the Collection is made with the Users need.

```G1ANT
dialog message ‴Testing LINE Android‴
lineandroid.open
delay 5
lineandroid.login phone ‴1234567892‴
delay 5
lineandroid.collections text ‴Collections section‴
```
