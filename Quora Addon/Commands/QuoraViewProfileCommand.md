# quora.viewprofile

## Syntax

```G1ANT
quora.viewprofile
```

## Description

# Sequence of Operation
The user is supposed to have an existing account on Quora
1. The user is allowed to open Quora.
2. The userid and password are being fed as input for which the validation takes place prompting the user to successfully login to Quora.
3. The program then shows the user his profile and keeps him updated with his profile

## Example

This simple script logs into Quora and views the user's profile

```G1ANT
dialog message ‴Testing out Quora‴
quora.open
delay 10
dialog message ‴Quora is open‴
quora.login email ‴test1@gmail.com‴ pword ‴Password‴ timeout 5
delay 10
quora.viewnotification 
dialog message ‴You are updated!‴
quora.viewprofile timeout 5
```
